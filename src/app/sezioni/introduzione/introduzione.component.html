1. Introduzione & Setup Ambiente

📌 Che cos’è Angular?

Angular è un framework per lo sviluppo di applicazioni web single-page (SPA) basato su TypeScript. È mantenuto da Google ed è pensato per costruire applicazioni modulari, scalabili e reattive.

⚙️ Requisiti

  • ✅ Node.js e npm
  • ✅ Angular CLI (`npm install -g @angular/cli`)

📦 Comandi base

ng new nome-progetto       # crea un nuovo progetto
ng serve                   # avvia il server di sviluppo
ng generate component xyz  # genera un nuovo componente
